```

```

```

특히 단기(1-3 개월), 중기(6-12개월), 장기(1년이상)로 나누어서 각각 종목 5개씩 설명해줘
```

```
2. 중기 (6~12개월)
3. 장기 (1년 이상)
각 종목·ETF마다 (1) 티커, (2) 선정 사유(섹터/실적 근거), (3) 리스크 요인을 간략히 포함
```

```지침
# 슬라이드 프롬프트
다음 조건을 모두 반영한 파워포인트 슬라이드 자료를 작성해줘.
- 전체 스타일은 현대적 요소와 전통적 요소가 결합된 학술 포스터 느낌으로 구성해줘.
- 연한 회색의 단색 배경을 사용하고, 선지 질감이 은은하게 보이게 해줘.
- 보라색 네온 계열의 포인트 컬러를 활용하되, 전체적으로는 여백이 넓고 정돈된 느낌을 유지해줘.
- 3D 부조감, 임파스토 질감, 미세한 입체 풍경 같은 비주얼 요소를 슬라이드의 분위기 요소로 반영해줘.
- 메인 타이틀은 눈에 띄는 예술적 캘리그래피 스타일로 배치해줘.
- 서브타이틀과 본문은 세리프 계열과 가독성 높은 폰트를 조화롭게 사용해줘.
- 모든 내용은 읽기 쉽고, 시각적 위계가 분명하게 보이도록 구성해줘.

# 슬라이드 구성
## 각 슬라이드는 다음 순서로 작성해줘.
- 제목과 함께 다음 중에는 이 페이지를 설명하는 한 문장.
## 블릿 목록과 블릿 규칙
- 블릿은 2~3 수준으로 구성해줘.
- 수준 1 블릿은 3~4개로 작성해줘.
- 각 수준 1 블릿 아래에는 수준 2 블릿을 2~4개씩 넣어줘.
- 필요하면 수준 3 블릿도 사용하되, 너무 복잡하지 않게 유지해줘.
- 블릿 기호는 수준마다 서로 다르게 구분해줘.
- 각 블릿은 짧고 명확한 문장으로 작성해줘.
```
