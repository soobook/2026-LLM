LLM과 AI Agent 입문
담당교수: 홍길동 교수 강의실: 공학관 301호 강의시간: 화·목 10:30–12:00
과목 개요
본 과목은 대규모 언어 모델(LLM)의 작동 원리와 AI Agent의 설계·구현을 체계적으로 학습한다. Transformer 아키텍처의 핵심 개념부터 프롬프트 엔지니어링, RAG, 멀티에이전트 시스템까지 이론과 실습을 균형 있게 다룬다. 수강생은 Python 기반 실습을 통해 실제 LLM API를 활용한 애플리케이션을 개발하고, 최종 팀 프로젝트에서 독자적인 AI Agent 시스템을 설계·발표한다. 급변하는 AI 기술 환경에서 현장 적응력을 갖춘 인재를 양성하는 것을 목표로 한다.
학습 목표
01LLM의 구조와 학습 원리를 이해하고 Transformer 아키텍처를 설명할 수 있다
02프롬프트 엔지니어링 기법을 활용해 LLM 성능을 최적화할 수 있다
03RAG 시스템을 구축하고 외부 지식과 LLM을 연동할 수 있다
04LangChain 등 프레임워크로 AI Agent를 설계·구현할 수 있다
05AI 윤리와 안전성 이슈를 인식하고 책임 있는 AI 시스템을 설계할 수 있다
